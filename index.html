📁 Projects

Detection and Analysis of Deepfake Manipulated Data (Engineering Design II Graduation Project)

An end-to-end deep learning framework was developed to identify AI-generated multimedia manipulations using advanced computer vision and digital forensics principles. Utilizing the benchmark FaceForensics++ dataset, the system's robustness was systematically evaluated across multiple dimensions:

  • 4 Manipulation Methods Analyzed: The framework effectively detects Deepfakes (Autoencoder-based face swapping), Face2Face (Real-time expression reenactment), FaceSwap (Graphics-based 3D face replacement), and NeuralTextures (GAN/Neural-driven patch regeneration).
  • 4 Robust Test Scenarios: To assess real-world generalization, the model was trained and validated across four pipeline variations: Full Frame, Face Crop (aligned/cropped facial regions via MTCNN), Masked areas, and Blurred frames.
  • Preprocessing & Architecture: MTCNN (Multi-task Cascaded Convolutional Networks) was utilized for optimal face detection. A custom-tuned EfficientNet-B0 model was trained from scratch via transfer learning. The entire data pipeline and hyperparameter tuning were built using Python and PyTorch within Google Colab GPU environments.
  • Advanced Evaluation: Video-level classification decisions were optimized using a majority voting algorithm. A comprehensive ablation study was conducted across various frame configurations (1, 5, 10, 50 frames), and confusion matrices were extracted to validate the model's exceptional generalization and detection performance.
Smart Baby Monitor (Embedded System Project)

Designed and deployed a microcontroller-based, real-time audio acquisition and wireless broadcasting system. Both hardware and software pipelines were fully built from scratch:

  • Hardware Design: Designed 2 custom electronic printed circuit boards (Transmitter and Receiver units) using Altium Designer, executing full schematic capture, layout optimization, and physical PCB fabrication.
  • Sensors & Signal Processing: Ambient sound levels were captured via an analog microphone sensor. High-efficiency data conversion was achieved by leveraging the microcontroller's ADC Interrupt (Analog-to-Digital Converter) routine to process incoming signals dynamically.
  • Wireless Communication: Processed telemetry and critical voice triggers were broadcast securely to the receiver unit utilizing an integrated Bluetooth module.
  • Control Mechanism: The receiver unit analyzed incoming real-time packages to automatically trigger a dynamic hardware alert system (LED array control) once customizable thresholds were exceeded.
